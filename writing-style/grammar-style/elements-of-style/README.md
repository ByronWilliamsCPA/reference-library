## How EoS Fits the Authority Hierarchy

EoS is the **Tier 1 drafting baseline**. It provides strong general principles that should
inform initial drafting. When CMS and EoS conflict, CMS governs. See `../cross-reference.md`
for documented conflicts.

EoS is especially useful for:

- Composition principles (active voice, omit needless words, positive form)
- Instilling economy and directness in prose
- Checking structure (one paragraph, one topic)

EoS is less reliable for:

- Punctuation rules — CMS is more detailed and authoritative
- Grammar edge cases — CMS ch. 5 provides more nuanced guidance
- Modern usage — some EoS entries are outdated or disputed
